flyPanels - an off canvas menu panel system for responsive websites

FlyPanels is a responsive off canvas panel plugin for websites or web apps. It supports all modern browsers. It can be used for menus, contact forms or anything you like really. Since version 2.0 this plugin is written in pure vanilla javascript and has no dependencies.

Try it out by clicking the hamburger icon or the search icon to the top left and right

Compared to many other off canvas menu plugins out there this one is more solid and behaves more like a native solution. Try it!

View on Github

(Download the source and read the documentation)

bower install flypanels --save
